COVID-19 Early Case Trend Analysis & Recovery Insights

Project completed using Python, Pandas, and Matplotlib.

Folder Structure:

* code : Python source files
* dataset : Input dataset
* charts : Generated visualizations
* reports : Project summary and findings

Key Analyses Performed:

* Demographic Analysis
* Infection Source Analysis
* Recovery Analysis
* Contact Exposure Analysis
* Correlation Analysis
* Regional Analysis

Generated Charts:

* Age Distribution
* Gender Distribution
* Recovery Distribution
* State Distribution
* Region Distribution
* Top Regions

Outcome:
The project analyzes early COVID-19 case trends, recovery patterns, infection sources, and regional impact to provide actionable public health insights.
